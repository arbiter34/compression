compression
===========

## Description ##

Basic implementation of huffman coding in C with a Command Line Interface.  

## How to Compile ##

 -**Run:**: ./make

## Usage ##

<b>compression [options] [source] [destination]</b>
	<br><br>Options:<br><br>
		-**-c**:			Compress<br><br>
		-**-d**:			Decompress
